# Feature 189: Kojo Ability/Talent Branch Systematic Review

**Status**: PROPOSED
**Type**: kojo
**Priority**: Medium
**Version**: v0.7+

---

## Background

### 発端となった議論

口系コマンド（フェラチオ等）の調査中に以下が判明:

1. **精液中毒（ABL:31）** は口系コマンドの実行判定に影響する（×3倍率）
2. しかし**口上分岐には使用されていない**
3. 同様に **Ｍ感度（TALENT:109）**、**口淫経験（EXP:25）** も蓄積されるだけで口上に反映されない

### 現状の問題

| 変数 | システム利用 | 口上分岐 | 状態 |
|------|:------------:|:--------:|:----:|
| ABL:精液中毒 | ○（実行判定） | × | **未活用** |
| TALENT:Ｍ感度 | × | × | **未活用** |
| EXP:口淫経験 | ○（蓄積） | × | **未活用** |
| ABL:Ｖ感覚 | ○ | × | **未活用** |
| TALENT:Ｖ感度 | ○ | × | **未活用** |
| ... | ... | ... | ... |

現行口上は **TALENT 4分岐（思慕なし/思慕/恋慕/恋人）** のみで、感覚・感度・中毒・経験による分岐がない。

---

## 目的

全COMカテゴリに対し、口上で分岐すべきABL/TALENT/EXPを特定し、テンプレートに追加する。

## 実装計画

### Phase 1: テンプレート更新

`Game/templates/KOJO_KX.ERB` に分岐パターン追加:

```erb
;-- 精液中毒分岐（口系COM用） --
IF ABL:精液中毒 >= 3
    ; 中毒描写
ELSEIF ABL:精液中毒 >= 1
    ; 軽度嗜好描写
ELSE
    ; 通常描写
ENDIF

;-- 感度分岐（汎用） --
IF TALENT:{部位}感度 >= 1
    ; 敏感描写
ELSEIF TALENT:{部位}感度 <= -1
    ; 鈍感描写
ELSE
    ; 通常描写
ENDIF
```

### Phase 2: 80番台（口系）への適用

v0.7 の口上作成時に以下を適用:
- COM_81 フェラチオ: 精液中毒 + Ｍ感度 + 口淫経験
- COM_82 シックスナイン: 同上
- COM_85 イラマチオ: 同上

### Phase 3: 60番台（挿入系）への適用

v0.7 の口上作成時に以下を適用:
- Ｖ感覚/Ｖ感度/Ｐ感度/Ｖ性交経験

### Phase 4: 既存口上のレトロフィット（オプション）

完了済み口上への分岐追加（v1.1+ で検討）
