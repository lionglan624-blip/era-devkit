## Implementation Notes

### Interactive Mode Architecture

```
InteractiveRunner
├─ Initialize (load game, set up character)
├─ ReadLine loop
│   ├─ Parse JSON command
│   ├─ Execute (call/set/dump/reset)
│   ├─ Capture differential output
│   └─ Write JSON response
└─ Cleanup on exit/error
```

### Coverage Tracking

Hook points:
- `ProcessState.IfStatement()` - Track IF branch taken
- `ProcessState.SelectCase()` - Track SELECTCASE branch
- Store: `{file, line, condition, hit}` per branch

### Process-Level Parallel

```csharp
// Worker process
dotnet run --project uEmuera.Headless.csproj -- . --unit test.json --output-mode json

// Parent orchestrator
foreach (test in tests.AsParallel().WithDegreeOfParallelism(N))
{
    var process = Process.Start(...);
    var result = process.WaitForExit() + ParseOutput();
}
```
