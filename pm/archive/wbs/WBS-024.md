## Implementation Notes

### Task 1: CallGraphAnalyzer

Extract and adapt call graph logic from DeadCodeAnalyzer:
- Reuse `ExtractCallTargets()` and `ExtractFunctionName()`
- Build bidirectional graph: callers (A→B) and callees (B←A)
- DOT output format:
  ```dot
  digraph callgraph {
    rankdir=LR;
    node [shape=box];
    "FUNC_A" -> "FUNC_B";
    "FUNC_A" -> "FUNC_C";
  }
  ```

### Task 2: LinterOptions

Add new options:
```csharp
public bool CallGraph { get; set; }
public string? CallGraphRoot { get; set; }
public int? CallGraphDepth { get; set; }
```

### Task 3: Program.cs

Add callgraph mode detection:
```csharp
if (args.Length > 0 && args[0] == "callgraph")
{
    return RunCallGraph(options);
}
```
