## Design

### Interface Definition (Final)

```csharp
internal interface IProcessInitializer
{
    bool InitializeParserAndCheckConfig(EmueraConsole console);
    bool LoadResources(EmueraConsole console);
    void LoadKeyMacros(EmueraConsole console);
    bool LoadReplaceFile(EmueraConsole console);
    void LoadRenameFile(EmueraConsole console);
    GameBase LoadGameBase(string csvDir, EmueraConsole console);
    ConstantData LoadConstantData(string csvDir, EmueraConsole console, bool displayReport);
    bool LoadHeaderFiles(string erbDir, bool displayReport, EmueraConsole console,
        IdentifierDictionary idDic, Process process);
    bool LoadErbFiles(string erbDir, bool displayReport, EmueraConsole console,
        ExpressionMediator exm, Process process, LabelDictionary labelDic);
    bool LoadErbFilesFromList(List<string> files, EmueraConsole console,
        ExpressionMediator exm, Process process, LabelDictionary labelDic);
}
```

### Implementation Strategy

1. Extract pure loading logic (no state mutation)
2. Process.Initialize() calls initializer methods
3. Process assigns returned objects to fields
4. Side effects (GlobalStatic registration) stay in Process
