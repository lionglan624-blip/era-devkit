## Implementation Notes

- Interface uses fontName + fontSize + style as cache key (3-level dictionary)
- Config.FontManagerInstance property allows DI injection
- Null assignment resets to default FontManager (same pattern as WarningCollector)
- Existing 14 files using Config.Font/GetFont/ClearFont unchanged (backward compatible)
