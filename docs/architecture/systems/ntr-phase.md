# ゲーム設計（現状/目標仕様）

## Phase（縦軸：進行段階）※推奨改良版
- Phase0: 基盤（主人公との日常/信頼/満足＝守る価値の可視化）
- Phase1: 外圧（脅迫/証拠/借り/環境圧）
- Phase2: 親しみ（デート・二人の時間・秘密・習慣化）
- Phase3: 接触（身体的境界線の移動：手→肩/髪→距離ゼロ→抱擁…）
- Phase4: 不可逆点ハブ（自己物語が書き換わる。理由別に分岐）
- Phase5: 性的関係（反復・習慣化・隠蔽スキル化）
- Phase6: 完堕ち（価値観/優先順位の転換。主体性が変わる）
- Phase7: 拡散（選択的拡散 / 商業化〔売春・接待など世界観置換〕）
- Phase8+: 主人公後日談（寝取り返し/取引/共犯/破滅/解放 等）
※「売春＝誰とでも」は直結させず 7A選択的拡散 / 7B商業化 で分けるのが望ましい。

## Phase遷移ルール
- Phase遷移は「パラメータ閾値到達」だけで自動にしない。
- 推奨：閾値（状態条件）+ OPP（機会）+ ハブイベント（象徴出来事）で遷移。
  - 例：同マップは進行値↑ではなく OPP（接触機会）↑のみ。
  - OPPを消費して訪問者が手を打ち、成功時にパラメータが動く。

# ルート（横軸：Phase4不可逆点の理由）
「本気不倫」と「主人公が一番好きなのに体を許す」を両立したい。
Phase4を理由別ハブとして複数ルートへ分岐させる。

推奨ルート（R1〜R6）:
- R1 脅迫・犠牲: 守るため／逃げ場なし（LOVE_MC維持しやすい）
- R2 馴化・流れ: 境界崩壊／空気／高揚（LOVE_MC維持しやすい）
- R3 逃避・麻酔: ストレス/孤立の避難（LOVE_MC維持しやすい）
- R4 取引・契約: 条件交換（LOVE_MC維持しやすいが自尊心コスト）
- R5 反発・復讐: 主人公の失点反動（本気不倫に寄りやすい）
- R6 本気不倫: 訪問者への恋愛化（LOVE_MCが落ちる/逆転し得る）

※R6は条件解禁推奨（主人公失点の累積 + 誘惑燃料/相性突出）。

# パラメータ設計（最小セット）
狙い：セックスに入っても主人公LOVEを自動上書きしない。上書きが起こるのは本気不倫（R6）など条件時のみ。

【訪問者→ヒロイン（ヒロイン×訪問者ごと）】
- LEV: 弱み/証拠（Evidence台帳の合成値。脅迫は「証拠生成/発見/作成」で増える）
- FAM: 馴化（近さに慣れ、境界線が溶ける）
- DEP: 依存/借り（救済・支援・穴埋め）
- TEM: 誘惑燃料（欲求不満/退屈/スリル×相性）
- SEXB: 身体習慣/連想（恋愛ではない）
- EMO: 訪問者への恋愛化（本気不倫の核。条件解禁でのみ伸びやすく）
- SUS/EXPOSE: 疑念/露見リスク（覗き・阻止・バレイベントの基盤）
- OPP: 機会（同マップ/同席で増える。進行値ではない）

【主人公↔ヒロイン（主人公側）】
- LOVE_MC: 主人公への恋愛好意（上書きされにくい軸）
- TRUST_MC: 主人公への信頼/安心
- SAT_MC: 主人公満足（会話/関係の質。FRUSTに影響）
- RES: 抵抗（キャラ基礎 + 主人公関係 + 欲求/退屈/スリル等で上下）
  - RESは関係値（恋人/人妻はベース高） + 会話量/性交満足/好感などで増減

【補足】
- 単純接触による「親密化」は好感度(AFF)とは別に FAM（馴化）として扱うのが望ましい。
- 訪問者魅力（顔/体型/匂い/相性）は作業爆発を避け、ATTR+タグ一致でCOMPAT（相性）としてTEMに乗せる方針。

# 身体接触の設計（Phase3以降の深化）
- 身体は「部位」ではなくゾーンで管理（公的→準私的→私的→禁域）。
- 進行は「触れたか」より「言い訳が崩れる」過程（名目/時間/秘匿/主導のタグで意味が変わる）。
- 手が触れる（事故）→口実（整える）→継続接触（慰め）→私的空間（密室）→境界テスト→不可逆点（拒否の言語が消える/秘密共同体化）。

# 露見（バレ）設計
- 露見は段階化：小露見（Phase2〜3）→中露見（Phase4〜5）→大露見（Phase5〜6）。
- バレ方タイプ：
  - 覗き成功（主人公が証拠確保）
  - 第三者目撃（館内・人里など）
  - 訪問者が匂わせ（支配・孤立）
  - ヒロインが崩れて自白
- 露見後は POST_ROUTE として別レイヤで分岐（修復/共犯/移籍/堕落固定）。

---

# 完堕ち（Phase6）の重要仕様
- 完堕ちは行為ではなく「価値観/優先順位/自己物語の固定化」。
- Phase6で暴露（主人公にバラす）も有力。
- LOVE_MCが0になっても良いルートを許容（R6/R5など）。

# 再構築（救済ルート：主人公が再び関係を結ぶ）
ユーザー希望：③純粋に再構築。
前提：館内公然レベルで破局・離縁済、ヒロインは訪問者側の恋人/妻/従属/接待役などに固定された状態から、主人公が再接続する。

推奨：NTR Phaseとは別に Rebuild軸（RB0〜RB5）を持つ。
- RB0 断絶
- RB1 接触許可（責めない/詮索しない/支援で TRUSTを稼ぐ）
- RB2 会話の習慣化
- RB3 二人の時間復活（制約付き）
- RB4 関係の再定義（秘密/公認など）
- RB5 救済エンド群
最小パラメータ：AFF_MC（好感）、TRUST_MC、SAFETY（妨害/制裁リスク）、AGENCY（主体性）。
RB3解禁は AFF_MCではなく TRUST_MC + AGENCY を要求する（救済の説得力）。
