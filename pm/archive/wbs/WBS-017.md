# WBS-017: ParserMediator Instance-Based

## Overview

Convert ParserMediator from static class to injectable service with backward-compatible static facade.

**Strategy**: Similar to Feature 012 (WarningCollector) - keep static facade methods that delegate to injectable instance via GlobalStatic.

## Implementation Notes

### Interface Design

```csharp
internal interface IParserMediator
{
    IWarningCollector WarningCollector { get; set; }
    Dictionary<string, string> RenameDic { get; }
    bool HasWarning { get; }

    void Initialize(EmueraConsole console);
    void ConfigWarn(string str, ScriptPosition pos, int level, string stack);
    void Warn(string str, ScriptPosition pos, int level);
    void Warn(string str, ScriptPosition pos, int level, string stack);
    void Warn(string str, LogicalLine line, int level, bool isError, bool isBackComp);
    void Warn(string str, LogicalLine line, int level, bool isError, bool isBackComp, string stack);
    void LoadEraExRenameFile(string filepath);
    void ClearWarningList();
    void FlushWarningList();
}
```

### Static Facade Pattern

Keep original `ParserMediator` class name but make it delegate to `GlobalStatic.ParserMediatorInstance`:

```csharp
internal static class ParserMediator
{
    public static void Warn(...) => GlobalStatic.ParserMediatorInstance.Warn(...);
    // ... other methods
}
```

This maintains backward compatibility with all 100+ call sites.
