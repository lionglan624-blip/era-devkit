## Implementation Notes

### Hook Points

Coverage tracking will hook into three instruction classes:

1. **IF_Instruction.DoInstruction** (Instraction.Child.cs:1842)
   - Loop evaluates `IfCaseList` conditions
   - Record each IF/ELSEIF condition and result

2. **SIF_Instruction.DoInstruction** (Instraction.Child.cs:1798)
   - Single-line IF evaluation
   - Record condition and result

3. **SELECTCASE_Instruction.DoInstruction** (Instraction.Child.cs:1891)
   - Loop evaluates `CaseExps` conditions
   - Record each CASE and whether it matched

### CoverageCollector Design

```csharp
public class CoverageCollector
{
    public class BranchInfo
    {
        public string File { get; set; }
        public int Line { get; set; }
        public string Condition { get; set; }
        public bool Hit { get; set; }
    }

    // Thread-local for concurrent execution
    [ThreadStatic]
    private static CoverageCollector _instance;

    public static bool IsEnabled => _instance != null;
    public static CoverageCollector Current => _instance;

    public void RecordBranch(string file, int line, string condition, bool taken);
    public string ToJson();
}
```
