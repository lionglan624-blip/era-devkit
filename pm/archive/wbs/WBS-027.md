# WBS-027: GlobalStatic DI Batch 1

Feature: [feature-027.md](feature-027.md)

## Summary

GameBase/Process area DI migration:
- GlobalStatic.Console (25 usages) → IEmueraConsole + ConsoleInstance DI
- GlobalStatic.MainWindow (7 usages) → IMainWindow + MainWindowInstance DI

**Implementation Strategy**: Add DI infrastructure (interfaces + DI properties) without requiring immediate migration of all usages. Existing code continues to work via fallback to static fields. Tests can inject mocks.
