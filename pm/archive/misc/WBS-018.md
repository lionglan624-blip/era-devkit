# WBS-018: Config Options Pattern

## Overview

Convert static Config class to injectable service with IConfigReader interface and static facade for backward compatibility.

**Strategy**: Similar to Feature 017 (ParserMediator) - create interface, service class, add to GlobalStatic, keep static facade.

## Implementation Design

### IConfigReader Interface

```csharp
internal interface IConfigReader
{
    // Core encoding
    Encoding Encode { get; }
    Encoding SaveEncode { get; }

    // Case sensitivity
    bool ICFunction { get; }
    bool ICVariable { get; }
    StringComparison SCFunction { get; }
    StringComparison SCVariable { get; }

    // Feature flags
    bool UseRenameFile { get; }
    bool UseReplaceFile { get; }
    bool UseMouse { get; }
    // ... ~60 more properties

    // Methods
    string GetConfigName(ConfigCode code);
    Font GetFont(string fontName, FontStyle style);
    void ClearFont();
    List<KeyValuePair<string, string>> GetFiles(string rootdir, string pattern);
    bool CheckUpdate();
}
```


### Static Facade (Config class)

```csharp
internal static class Config
{
    // Delegate all property access to GlobalStatic.ConfigInstance
    public static Encoding Encode => GlobalStatic.ConfigInstance.Encode;
    public static Encoding SaveEncode => GlobalStatic.ConfigInstance.SaveEncode;
    // ...

    // Initialization methods call through to instance
    public static void SetConfig(ConfigData instance)
        => GlobalStatic.ConfigInstance.SetConfig(instance);
}
```

### GlobalStatic Addition

```csharp
private static IConfigReader _configInstance;
public static IConfigReader ConfigInstance
{
    get => _configInstance ?? (_configInstance = new ConfigService());
    set => _configInstance = value;
}

// For initialization access (SetConfig, etc.)
internal static ConfigService ConfigServiceInstance
    => ConfigInstance as ConfigService ?? throw new InvalidOperationException();
```
