# M絶頂/五重絶頂システム

**Status**: DRAFT
**Version**: v1.8
**Source**: eraTW analysis

---

## 概要

口（Mouth）への快感を追跡する快Ｍパラメータと、それに伴うM絶頂・五重絶頂システムの追加。

### 現状（紅魔館protoNTR）

| 項目 | 値 |
|------|-----|
| 快感部位 | C, V, A, B（4種） |
| 最大多重絶頂 | 四重絶頂 |
| 無自覚絶頂 | なし |

### 目標（eraTW準拠）

| 項目 | 値 |
|------|-----|
| 快感部位 | C, V, A, B, **M**（5種） |
| 最大多重絶頂 | **五重絶頂** |
| 無自覚絶頂 | **5種（C/V/A/B/M）** |

## 実装計画

### Phase 1: CSV定義追加

| ファイル | 変更内容 |
|----------|----------|
| `palam.csv` | 4,快Ｍ 追加 |
| `source.csv` | 4,快Ｍ / 44,与快Ｍ 追加 |
| `Juel.csv` | 4,快Ｍ 追加 |
| `ex.csv` | 7,Ｍ絶頂 / 8,五重絶頂 / 20-24,無自覚絶頂 追加 |
| `exp.csv` | Ｍ絶頂経験 追加 |

### Phase 2: 絶頂ロジック改修

**TRACHECK_ORGASM.ERB** の `@ORGASM_ADD` 関数を改修:

```erb
; 現状: 4部位判定
絶頂候補 += ((CUP:奴隷:快Ｃ + PALAM:奴隷:快Ｃ) >= PALAMLV:4)
絶頂候補 += ((CUP:奴隷:快Ｖ + PALAM:奴隷:快Ｖ) >= PALAMLV:4)
絶頂候補 += ((CUP:奴隷:快Ａ + PALAM:奴隷:快Ａ) >= PALAMLV:4)
絶頂候補 += ((CUP:奴隷:快Ｂ + PALAM:奴隷:快Ｂ) >= PALAMLV:4)

; 追加: 快Ｍ判定
絶頂候補 += ((CUP:奴隷:快Ｍ + PALAM:奴隷:快Ｍ) >= PALAMLV:4)
```

**多重絶頂判定の拡張**:

| 種類 | 条件 | 倍率 |
|------|------|:----:|
| 五重絶頂 | C+V+A+B+M全て | 16倍 |
| 四重絶頂 | 5C4=5通り | 8倍 |
| 三重絶頂 | 5C3=10通り | 4倍 |
| 二重絶頂 | 5C2=10通り | 2倍 |

**組み合わせ数の増加**:

| 多重絶頂 | 現状（4部位） | 拡張後（5部位） |
|----------|:-------------:|:---------------:|
| 四重 | 1通り | 5通り |
| 三重 | 4通り | 10通り |
| 二重 | 6通り | 10通り |
| **合計** | **11通り** | **25通り** |

### Phase 3: コマンド連携

快Ｍを上昇させるコマンド（口系）:

| COM# | 名前 | 快Ｍ上昇 |
|:----:|------|:--------:|
| 80 | フェラ | ○ |
| 81 | 素股フェラ | ○ |
| 82 | シックスナイン | ○ |
| 83 | 口に出す | ○ |
| 84 | 深淵を覗く | ○ |
| 85 | イラマチオ | ○ |
| 20 | キスする | △（軽微） |

### Phase 4: 口上対応

**口上テンプレート追加**:

```erb
;-- M絶頂口上 --
@KOJO_M_ORGASM_{N}
    ; Ｍ絶頂時の口上（口で絶頂）

;-- 五重絶頂口上 --
@KOJO_QUINTUPLE_ORGASM_{N}
    ; 五重絶頂時の口上（全身絶頂）
```

### Phase 5: 無自覚絶頂（オプション）

快感を自覚せずに絶頂する状態。TALENTや状態による分岐。

```erb
; 無自覚絶頂の条件例
IF TALENT:奴隷:自制心 && EXP:奴隷:絶頂経験 < EXPLV:3
    ; 絶頂したが自覚なし
    NOWEX:奴隷:無自覚Ｍ絶頂 = 1
ENDIF
```
