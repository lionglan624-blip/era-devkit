### QUICK_START_SETUP function

New function to set defaults and jump to SHOP:
1. Call DEFAULT_OPTION (already done before MODE_SELECT)
2. Add default characters (MASTER already exists, ADDCHARA 1-13)
3. Set character positions and clothing
4. Set starting date/time
5. SETBIT FLAG:ＮＴＲパッチ設定,0 (skip NTR options)
6. BEGIN SHOP
