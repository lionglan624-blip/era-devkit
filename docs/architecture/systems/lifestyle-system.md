## 1. システム概要

### 1.1 設計思想

「食事・運動・性行為」によるステータス成長システム。料理がビルドの核、運動が基盤、男性器ステータスは取り返しのつかない方向性を決定する。

| システム | 役割 | 複雑さ |
|----------|------|:------:|
| 食事・料理 | ビルドの核 | 高 |
| 運動 | 基盤・上限解放 | 低 |
| 男性器ステータス | 方向性決定 | 中 |
| 弾幕ステータス | 戦闘特化 | 中 |

### 1.2 ステータス相互関係

```
食事 ─┬→ 男性器ステ（サイズ/硬さ/量/濃さ）
      ├→ 弾幕ステ（精度/集中/持続）
      ├→ 体力/気力
      └→ 好感度（美味しさ）

運動 ─┬→ 体力最大値↑
      ├→ 弾幕ステ基盤
      └→ 男性器ステ上限解放

性行為 → 男性器ステ変動（短期・変動型）
```

## 2. 食事・料理システム

### 2.1 技能体系

#### 主要技能

| 技能 | 役割 | 上限 |
|------|------|:----:|
| 料理技能 | 調理成功率・効果倍率 | 10 |
| 食材知識 | 素材の見極め・隠し効果 | 10 |
| 魔力調理 | 魔力素材の扱い | 10 |
| 衛生管理 | 失敗・デバフ防止 | 5 |
| 交渉 | 購入価格・入手量 | 5 |

#### サブ技能

| 技能 | 効果 | 上限 |
|------|------|:----:|
| 包丁技 | 繊維処理・太さ系補正 | 5 |
| 火加減 | 硬さ・出力系 | 5 |
| 盛り付け | 美的完成度・好感度 | 5 |
| 発酵管理 | 集中力・遅効性 | 5 |

### 2.2 食材カテゴリ

#### A. 基礎食材（序盤）

| 食材 | 入手 | 価格 | 要求技能 | 難度 | 効果 |
|------|------|-----:|----------|:----:|------|
| 野菜 | 購入 | 50 | なし | ★ | 回復量↑、デバフ解除 |
| 米・パン | 購入 | 80 | なし | ★ | 体力↑ |
| 卵 | 購入 | 120 | 食材知識1 | ★ | 量↑、濃さ↑、精力↑ |

**設計意図**: ステ上昇控えめ、「美味しさ」重視で好感度が伸びやすい

#### B. 精力系食材（中盤）

| 食材 | 入手 | 価格 | 要求技能 | 難度 | 効果 |
|------|------|-----:|----------|:----:|------|
| 赤肉 | 購入 | 300 | 包丁技1 | ★★ | サイズ↑、硬さ↑、体力↑ |
| 鶏肉 | 購入 | 220 | 火加減1 | ★★ | 太さ↑、体力↑ |
| 乳製品 | 購入 | 180 | 衛生管理1 | ★★ | 太さ↑、性感↑ |

**効果傾向**: 体力/基礎ステ上昇、調理失敗で疲労デバフ

#### C. 集中・弾幕系食材（中盤〜後半）

| 食材 | 入手 | 価格 | 要求技能 | 難度 | 効果 |
|------|------|-----:|----------|:----:|------|
| 魚介 | 購入/釣り | 350 | 食材知識2 | ★★★ | 弾道精度↑、集中力↑、硬さ↑、濃さ↑ |
| 発酵素材 | 自作 | 200 | 発酵管理2 | ★★★ | 集中力↑（遅効） |
| 苦味薬草 | 採取 | 非売 | 食材知識2 | ★★★ | 精神安定↑ |

**効果傾向**: 弾幕精度・集中力、即効性低/遅効性高

#### D. 魔力素材（後半）

| 食材 | 入手 | 価格 | 要求技能 | 難度 | 効果 |
|------|------|-----:|----------|:----:|------|
| 魔力結晶 | 戦闘 | 非売 | 魔力調理2 | ★★★★ | 弾幕全般↑、汚染度↑ |
| 紅魔素材 | イベント | 非売 | 魔力調理3 | ★★★★★ | 紅魔補正↑、濃さ↑↑ |
| 異界調味料 | 取引 | 800 | 交渉2 | ★★★★ | 特殊効果 |

**効果傾向**: 弾幕特化/汚染度増、失敗時は暴発・混乱

### 2.3 使用可能食材数

| 料理技能Lv | 使用可能食材数 |
|:----------:|:--------------:|
| 0-1 | 1個 |
| 2-3 | 2個 |
| 4-5 | 3個 |
| 6+ | 4個（上限） |

**魔力素材**: 常に1枠固定（他素材を圧迫）

#### 効果計算

| 素材位置 | 効果率 |
|----------|:------:|
| 主素材 | 100% |
| 副素材1 | 60% |
| 副素材2 | 30% |
| 副素材3 | 10% |

**設計意図**: 素材数を増やすほど「味・雰囲気・好感度」は伸びやすいがステ効率は逓減

### 2.4 複合素材シナジー

| 組み合わせ | 効果 |
|------------|------|
| 肉＋乳 | 太さ・体力↑↑ |
| 魚＋苦味 | 弾道精度↑↑ |
| 発酵＋野菜 | 集中力（遅効）↑↑ |
| 魔力＋魚 | 弾幕密度↑、汚染度↑ |
| 甘味＋乳 | 気力↑、好感度↑↑ |

### 2.5 調理法

#### 調理法アンロック

| 調理法 | 必要技能 | 効果傾向 |
|--------|----------|----------|
| 焼く | 料理1 | サイズ・硬さ↑ |
| 煮る | 料理2 | 安定成長 |
| 発酵 | 発酵管理2 | 集中力（遅効） |
| 魔力煮 | 魔力調理2 | 紅魔補正↑、汚染度↑ |
| 儀式調理 | 魔力調理4 | 能力解放 |

#### 調理法による差

| 調理法 | 効果 | リスク |
|--------|------|--------|
| 生 | 濃さ↑↑ | 失敗率高 |
| 丁寧 | 美的完成度↑ | 時間 |
| 雑 | 出力↑ | 精度↓ |
| 即席 | 即効性↑ | 持続↓ |
| メイド調理 | 好感度補正↑ | - |

### 2.6 料理名スケーリング

実在料理名を使用し、技能・器具・素材で"格"が変わる。

#### 例: ムニエル系

| グレード | 条件 | 表示名 |
|----------|------|--------|
| 低 | 技能1 | 焼いた魚 |
| 中 | 技能2 | 魚のムニエル |
| 高 | 技能4＋バター | バター香るムニエル |
| 特 | 技能6＋器具 | 紅魔館風ムニエル |
| 禁 | 魔力調理 | 禁製ムニエル |

#### 料理カテゴリ

| 系統 | 料理例 |
|------|--------|
| 焼 | ステーキ、ソテー |
| 煮 | シチュー、ポトフ |
| 揚 | フリット |
| 卵 | オムレツ |
| 乳 | グラタン |
| 魚 | ムニエル、カルパッチョ |

### 2.7 調理器具

#### 器具一覧

| 器具 | 効果 | 入手 |
|------|------|------|
| 包丁 | 失敗率↓ | 購入 |
| フライパン | 焼系解放 | 初期 |
| 鍋 | 煮系解放 | 購入 |
| オーブン | 上位焼 | イベント |
| 発酵樽 | 発酵解放 | 自作 |
| 魔導調理器 | 魔力調理 | 後半 |

**設計意図**: 器具は「解放・安定・補正」用。数値インフレ源にしない

#### 器具の役割

- 調理法アンロック
- 失敗イベント防止
- 効果のブレ幅を小さくする

### 2.8 技能不足時のペナルティ

| 技能不足 | 起きること |
|----------|-----------|
| 料理技能不足 | 効果半減 |
| 食材知識不足 | 隠し効果無効 |
| 衛生不足 | デバフ付与 |
| 魔力調理不足 | 暴発イベント |

### 2.9 「上がらない」料理の重要性

**ステ補正なし料理の存在意義**:

| 特徴 | 効果 |
|------|------|
| ステ補正なし | 数値変化なし |
| 料理描写重視 | 会話増加 |
| 安心感 | 好感度上昇率↑ |
| 日常演出 | フラグ緩和 |

**設計意図**:
- ステータス育成最適解ではない
- しかし人間関係構築では最適
- 「堕ちていない・日常・戻れる世界」の象徴

## 3. 運動システム

### 3.1 設計方針

**運動は単純で良い**。料理より細かくしない。

**理由**:
- 料理: 準備・素材・技能 → 複雑でOK
- 運動: 日課 → 複雑だと作業化

### 3.2 運動カテゴリ（固定）

| 種類 | 効果 | リスク |
|------|------|--------|
| 軽運動 | デバフ防止 | なし |
| 基礎鍛錬 | 上限解放（体力/硬さ） | なし |
| 集中鍛錬 | 精度補正（弾幕） | 疲労 |
| 過剰鍛錬 | 一時上昇＋反動 | 性感↓ |

### 3.3 運動の役割

運動は「直接上げる」より「最大値キャップ解除」役

| 運動 | 男性器ステ | その他 |
|------|-----------|--------|
| 下半身鍛錬 | 硬さ↑ | 体力↑ |
| 全身運動 | 全体微増 | 最大値上限↑ |
| 魔力循環 | 濃さ↑ | 弾幕性能↑ |

### 3.4 料理技能との連動

料理技能が高いほど運動効率が微増（間接的に連動）

## 4. 弾幕ステータス

### 4.1 設計方針

- 「火力一択」にしない
- 料理・生活・精神状態が弾幕性能に反映
- 性的ステータスとは部分的に連動（完全一致しない）

### 4.2 コアステータス

| ステ名 | 説明 | 主な影響 |
|--------|------|----------|
| 弾幕出力 | 一発の威力 | ボス戦短縮 |
| 弾幕密度 | 弾の数・隙間 | 回避強要 |
| 弾速制御 | 速さの制御力 | 初心者殺し |
| 弾道精度 | 狙った所に飛ぶ | 命中率 |
| 持続力 | 長期戦耐性 | スペル維持 |
| 魔力回復 | リチャージ | 連戦可能 |
| 集中力 | 操作安定 | 被弾減少 |
| 精神安定 | 弾幕の乱れ防止 | 暴発防止 |

### 4.3 上級・隠しステータス

| ステ名 | 内容 |
|--------|------|
| 美的完成度 | 弾幕が美しいほど評価↑ |
| 威圧感 | 開幕時の圧 |
| 汚染弾性 | 精神干渉・催眠 |
| 対人圧 | 対人戦補正 |
| 紅魔補正 | 館特有の加護 |

### 4.4 弾幕ステが上がる要素

#### 食事

| 食材・料理 | 上がる弾幕ステ |
|------------|---------------|
| 魚介 | 弾道精度・集中力 |
| 血肉料理 | 弾幕出力 |
| 魔力素材 | 魔力回復・持続 |
| 苦味系 | 精神安定 |
| 発酵食品 | 集中力（遅効） |

#### 行動・生活

| 行動 | 効果 |
|------|------|
| 瞑想 | 集中力↑ |
| 夜更かし | 出力↑、精度↓ |
| 過労 | 全体↓ |
| 館内散策 | 精神安定↑ |

### 4.5 男性器ステとの連動

一部は男性器ステと連動:
- 濃さ↑ → 汚染弾性↑

## 5. 男性器ステータス成長

**詳細設計**: [ntr-flavor-stats.md](ntr-flavor-stats.md)

### 5.1 成長曲線

**ハイブリッド型（序盤線形 → 中盤以降指数）** を採用

| Lv | 必要経験値 |
|:--:|:----------:|
| 1→2 | 10 |
| 2→3 | 20 |
| 3→4 | 40 |
| 4→5 | 80 |
| 5→6 | 160 |
| 6→7 | 320 |

**理由**:
- 完全線形: 後半インフレ崩壊
- 完全指数: 序盤が苦行
- eraは「堕ちるほど重くなる」設計と相性が良い

**完堕ち後**: 上昇量↑ / 必要量↑↑（極端化するが止まる）

### 5.2 成長方法別効果

| 方法 | 効果対象 | 傾向 |
|------|----------|------|
| 食事 | 長期・安定型 | 全般的に微増 |
| 運動 | 上限解放 | 硬さ・持続力 |
| セックス | 短期・変動型 | 相手依存 |

### 5.3 食事による成長

| 食材 | 男性器ステ | その他 |
|------|-----------|--------|
| 赤肉 | サイズ↑、硬さ↑ | 体力↑ |
| 鶏卵 | 量↑、濃さ↑ | 精力↑ |
| 乳製品 | 太さ↑ | 性感↑ |
| 魚介 | 硬さ↑、濃さ↑ | 弾幕集中力↑ |
| 野菜 | 変化なし | 回復量↑ |
| 甘味 | なし | 気力↑↓ |

### 5.4 セックスによる変動

| 行為 | 効果 |
|------|------|
| 通常 | 微増・安定 |
| 連続 | 硬さ↓、量↓ |
| 寝取られ | サイズ↑、支配力↑ |
| 被支配 | 濃さ↑、汚染度↑ |
| 愛情 | 太さ↑、好感度↑ |
| 強制 | サイズ↑↓（ランダム） |
