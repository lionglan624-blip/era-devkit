## Technical Design

### セーブファイル構造

```
[Header]
  GameCode, Version, Timestamp, CharacterCount

[Character Data] × CharacterCount  ← ※グローバルより先
  NAME, CALLNAME (2 strings)
  ISASSI, NO (2 integers)
  BASE, MAXBASE, ABL, ... (17 arrays)

[Global Variables - Eramaker format]
  DAY, MONEY, ITEM, FLAG, ... (60 arrays)
  SAVESTR (1 string array)

[Emuera Extended Section]
  __EMUERA_1808_STRAT__
  key:value pairs
  __EMU_SEPARATOR__
  array data
  ...
```

### 実装方針

1. **SaveReader.cs拡張**
   - `ReadCharacters()`: キャラクターデータ読み込み
   - `ReadGlobalArrays()`: グローバル配列読み込み
   - 非ゼロ値のみ保存（スパース形式）

2. **Program.cs拡張**
   - `--header`: 既存（ヘッダーのみ）
   - デフォルト: 全データ出力
   - `--character`: キャラ名フィルタ
   - `--filter`: 変数フィルタ

### JSON出力形式

```json
{
  "file": "save.sav",
  "header": { ... },
  "emueraVersion": 1808,
  "globals": {
    "FLAG": {"26": 128, "100": 1}
  },
  "characters": [
    {
      "id": 4,
      "name": "十六夜咲夜",
      "callName": "咲夜",
      "arrays": {
        "CFLAG": {"297": 4}
      }
    }
  ]
}
```
