## 実装ガイドライン

### C2品質基準

**行数**: 4-8行（PRINTFORM単位）

**必須要素**:
1. **感情描写**: 内的反応（恥ずかしさ、快感、抵抗感など）
2. **情景描写**: 外的状況（身体接触、音、雰囲気など）
3. **好感度分岐**: TALENT 4段階（思慕以下、思慕、恋慕、恋人/人妻）
4. **キャラクター性**: 各キャラの固有表現

**参考品質**: v0.5完了COM（300-316）+ v0.6完了COM（0-72）

### TALENT分岐構造

```erb
IF TALENT:TARGET:関係度 >= 人妻判定値        ; 人妻
    PRINTFORM ...
ELSEIF TALENT:TARGET:関係度 >= 恋人判定値    ; 恋人
    PRINTFORM ...
ELSEIF TALENT:TARGET:関係度 >= 恋慕判定値    ; 恋慕
    PRINTFORM ...
ELSEIF TALENT:TARGET:関係度 >= 思慕判定値    ; 思慕
    PRINTFORM ...
ELSE                                        ; 思慕以下
    PRINTFORM ...
ENDIF
```
