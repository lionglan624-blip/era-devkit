## System Changes Required

### TCVAR:116 主導権分岐システム

**現状**: TCVAR:116（行為者）は設定されているが、読み取り処理なし（未使用）

**改修内容**:
- 口上で `IF TCVAR:116 == PLAYER` / `IF TCVAR:116 == TARGET` 分岐追加
- SOURCE処理での主導権による補正
- 挿入系コマンドで主導権選択UI追加（オプション）

**対象コマンド**:
- 挿入系: 正常位、後背位、騎乗位、座位系
- 奉仕系: フェラ、手淫、パイズリ等

**効果**:
- 60系/90系の重複解消可能
- 「突き上げる」「腰を振らせる」等の描写分岐
- 口上バリエーション倍増

### COMABLE2.ERB 制限緩和

**現状**: 310-399は日常モード専用（調教モードで使用不可）

**改修案**:
```erb
CASE 310 TO 326  ; 軽いスキンシップ系 → 両モードOK
CASE 327 TO 399  ; それ以外 → 日常限定維持
```

## Implementation Priority

### Phase 1: 低難度・高効果

1. **欠番復活** (3件): COM43, COM187, COM511
2. **300系スキンシップ** (11件): COMABLE2.ERB改修 + 新規
3. **200系脱衣** (2件): COM204, COM205

### Phase 2: 中難度

4. **60系体位** (6件): 既存パターン踏襲
5. **80系手技** (2件): 複合奉仕
6. **100/140系ハード** (2件): 異常経験連動

### Phase 3: 高難度・システム連動

7. **350系介入** (5件): NTRシステム連携
8. **400系日常** (4件): 判定ロジック設計必要
9. **TCVAR:116主導権** (システム): 全挿入/奉仕系に影響

## Notes

- COM番号は暫定。実装時にCSV空き番号を確認
- 口上は全て新規作成が必要
- 判定ロジック（成功/失敗）は個別設計が必要なコマンドあり
