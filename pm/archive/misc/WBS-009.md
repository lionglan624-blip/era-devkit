## Technical Design

### 関数仕様

**@INVAGINA(ARG)** - V（膣）の状態
```
TEQUIP:ARG:Ｖセックス → 1 (ペニス)
TEQUIP:ARG:バイブ → 2 (バイブ)
その他 → 0 (なし)
```

**@INANAL(ARG)** - A（アナル）の状態
```
TEQUIP:ARG:Ａセックス → 1 (ペニス)
TEQUIP:ARG:アナルバイブ → 2 (アナルバイブ)
TEQUIP:ARG:アナルビーズ → 3 (アナルビーズ)
その他 → 0 (なし)
```

### TEQUIP インデックス

| Index | Name |
|-------|------|
| 13 | バイブ |
| 14 | アナルバイブ |
| 15 | アナルビーズ |
| 50 | Ｖセックス |
| 51 | Ａセックス |

### テストシナリオ

| # | Scenario | TEQUIP設定 | Expected |
|---|----------|-----------|----------|
| 1 | V:ペニス | 50=1 | INVAGINA=1 |
| 2 | V:バイブ | 13=1 | INVAGINA=2 |
| 3 | A:ペニス | 51=1 | INANAL=1 |
| 4 | A:アナルバイブ | 14=1 | INANAL=2 |
| 5 | A:アナルビーズ | 15=1 | INANAL=3 |
| 6 | V+A同時 | 50=1, 51=1 | 両方表示 |
| 7 | 何もなし | 全て0 | 両方0 |
